GENERAL

The system consists of motrized blinds, connected to MCU’s (Motor Control Units) controlled by a central computer system.

The central computer system receivs commands from sensors and user input from wall panels, which are of two types: AMX and Triton panels.

The communication with the MCU’s is done through a BUS cable which sends serial commands, which are then translated into electrical signals.

The system is designed to communicate and integrate with other industry systems.

The system is controlled by a central computer, and it has 3 main software components which need to be running at all times.

Detailed description of the hardware and software components in the following sections.

HARDWARE

Each sealed unit has a motor controlling a rod in the top side, encased in an aluminium profile.

The motors are FAULHABER stepper motors, controlled by two wires.

The main motor wires are plugged in an MCU (motor control unit) placed on motherboards which provide a 24V power suply and a platform for the BUS connection.

The motherboards are connected to each other in series and then to a MOXA RS232 to USB converter to the central computer.

The central computer then send and receives information through the serial port.

A KNX serial connection is also used to receive signals from the different systems, from user input , sensor and timer commands.




SOFTWARE

The communication of the system to the motorized blinds is done through serial communication possible through a MOXA RS 232 to USB converter which connects to the data ports of the MCU’s.

The system is comprised of 3 main softwares which communicate with each other.


The Simbus program insures serial communication with the blinds, using configuration files which have the address of each blind programmed.



The WMA server is a listener to signals from third party systems, like the commands from sensors and wall panels.



A KNX OPC protocol provides the bridge between the previous softwares.



All the programs are set up to initiale automatically at the start of the system and they need to be running at all times for a correct function of the system.



OPERATION

The blinds can be operated through the Persienner App or the wall panels, which are of two types: AMX Panels and Triton Panels. The way these operate the blinds is different, and described below.


AMX Panels

TThe AMX Touchscreen panel has two buttons which have 3 functions. Once pressed the left button will change function. All buttons are disabled while the blinds are moving.


Triton Panels

The blinds can be operated through conventional button press panels, Triton Panels. The way these operate the blinds is different depending on rooms where blinds can lift or not.
